    fr.yogoko.choir.lib.asn1.Asn1ChoiceImpl Class Reference

    Detailed Description

    Represents an ASN.1 choice type.

    Each item for choice "alternative" has an identical index, from 0 to n-1.

    Alternative-to-index mapping shall be defined by implementer.
