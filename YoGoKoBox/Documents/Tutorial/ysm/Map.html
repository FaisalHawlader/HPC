    Map

    Map is the main element of this app. It displays a visual representation of the surrounding road environment. It can be configured to display only what users want with configuration Map.
    Map will display every ITS station within the radius defined in LdmService configuration, as well as every DENM event and traffic light if their relevance is of at least two.

    Note: Device on which the application is running must have internet connectivity to display the map.
