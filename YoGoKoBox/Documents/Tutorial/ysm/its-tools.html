YoGoKo Smart Middleware

    ITS Tools

    ITS Tools is a set of helper scripts that abstract low-level API calls. This allows a user to easily perform monitoring or interact with facilities and applications services, with a limited knowledge of how YoGoKo Smart Middleware works.

    List of ITS Tools commands

    This section provides a list and description of ITS Tools commands. For each tool, the equivalent API call from the Command-Line Interface API is provided. This allows a user to manually execute the corresponding command, and adapt it to its needs.

    Each tool can be called with -h to display usage information. For instance: 

    $ its-cam-received -h

Monitor received CAM messages
Usage: ./bin/its-cam-received [options]
Options:
 -h         Print this help message
 --ip=<IP>  Specify the middleware IP
 -q         Do not print information messages
Description:
This tool monitors CAM messages received by the CA Service. Messages
 are printed on stdout under a JSON format
